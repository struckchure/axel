# Data Types

The value types a [property](/asl/fields/properties) or
[parameter](/aql/parameters/typed) can hold.

- **[Scalars](/asl/data-types/scalars)** — built-in scalar types and their PostgreSQL mappings.
- **[Aliases](/asl/data-types/aliases)** — named aliases over a built-in scalar.
- **[Enums](/asl/data-types/enums)** — enumerated string types with a `CHECK` constraint.
