# Schema

The building block of a schema is the **type**. This section covers types
themselves and the schema-level declarations inside a type body. For the fields
inside a type, see [Fields](/asl/fields).

- **[Types](/asl/schema/types)** — concrete and abstract types.
- **[Inheritance](/asl/schema/inheritance)** — extending one or more types.
- **[Indexes](/asl/schema/indexes)** — `index on (...)` declarations.
- **[Constraints](/asl/schema/constraints)** — composite (type-level) constraints.
